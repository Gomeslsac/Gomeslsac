## 🧩 Projetos e áreas em destaque

### 🎟️ Sistema de Rifas
Projeto em **Laravel** com arquitetura MVC, pensado para uso real em hospedagem, com foco em qualidade de código, organização e escalabilidade. O sistema inclui autenticação, criação e gerenciamento de rifas, compra de números, controle de disponibilidade, painel administrativo e integração com pagamento via **Pix**.

### 📋 Sistema de Pesquisa
Estrutura de plataforma com lógica multiusuário e foco em gestão de pesquisas, usuários, entrevistadores, perguntas, respostas e relatórios. Projeto pensado com atenção tanto para a **arquitetura de dados** quanto para a **experiência visual**.

### 📱 Aplicação Mobile com React Native
Aplicativo desenvolvido com **React Native + Expo**, com foco em formulários, autenticação, contexto de conta, fluxo offline/online, armazenamento local e sincronização posterior.

### 👥 Sistema Web de Gestão de Clientes
Sistema em **PHP + MySQL**, com autenticação por sessão, CRUD completo, exportação para PDF, gráficos e melhorias planejadas como estrutura MVC, AJAX e exportação para CSV/Excel.

### 🤖 Automações e Integrações
Experimentos e implementações com **Docker**, **n8n**, **WAHA** e integrações com IA, focando em automação de processos, fluxos de mensagens e organização de serviços locais.
